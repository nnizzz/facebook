A Flutter-based Facebook UI clone built to practice and demonstrate modern Flutter widgets, layouts, and navigation.
This project focuses purely on UI replication and frontend concepts, not backend functionality.
Screens Implemented:
1.Login Screen
2.Home Screen
3.Stories Section
3.Profile Page
4.Notifications Screen
5.Chats UI
6.Friend Requests Screen
